# 反转链表

## 题目描述
输入一个链表，反转链表后，输出新链表的表头。

### 解题思路：
遍历: 将指向下一个节点的指针指向上一个节点。


### 代码：
***遍历法***

```

import java.util.Stack;
public class Solution {
    public ListNode ReverseList(ListNode head) {
        if(head == null){
            return head;
        }
        ListNode pre = head;
        ListNode curr = head.next;
        ListNode tmp = null;
        
        while(curr != null){
            tmp = curr.next;
            curr.next = pre;
            pre = curr;
            curr = tmp;
        }
        head.next = null;
        head = pre;
        return head;
    }
}
```
